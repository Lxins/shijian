数组

数组的length属性 不仅能读还能写，在设置数组length属性的时候
如果设置的length小于原来数组的长度 那么就会截取数组的前length个
如果设置的length大于原来的长度 那么原来的数组就会用对应个数的undefined去填充

如果只有一个参数 ，当这个参数是数字的时候，创建这个参数长度的数组，数组中的每一项都是undefined，
如果是大于一个参数，那么就是和[]是一样的。
如果参数是个0 那么就会创建一个 空数组

arr.indexOf(el,index) 用来查找数组中的某一项，如果找到了就返回对应位置，如果没找到就返回-1 从前向后查找 index 为可选参数，从第几位开始查找

arr.lastIndexOf(el,index) 是从后向前查找

arr.push(11,12,13 …) 向数组最后面添加新的元素，返回数组新的长度

arr.pop() 删除数组中最后一位，返回值是被删除的最后一位

arr.unshift(11,12,13…) 向数组最前面添加新的数据，返回数组新的长度

arr.shift() 删除数组中第一位，返回值是被删除的第一位

slice(start,end) 用来截取数组，包括start 不包括end，
如果 start 或者 end < 0 就等于 start + length 或者 end + length
如果start > end 就返回 [] ，其他情况返回被截取的数据

arr.splice(index,delete_num,[1l,l2,l3 …]) 用来删除或者替换数组中指定位置开始的多少个元素，返回值是被删除的那个数组(内的数据是被删除那个位置的数据)

Array.from(类数组) 把一个类数组转换成数组

Array.reverse() 用来反转数组

arr.forEach(function(item,i,arr){ }) 用来便利数组中的每一项 item 代表 数组中的每一项 i 代表每一项对应的下标 arr 代表数组本身

arr.map(fn) 使用方法和forEach(fn)一样，只不过它接受的函数参数可以有返回值

arr.fill(context,[start,end]); 用来填充数组，第一个参数是要填充的内容 后面两个可选参数，从哪里
