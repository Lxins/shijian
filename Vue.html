Vue

vue是什么

一个mvvm框架(库)
一片html代码配合上json，在new出来vue实例

优点

比较容易上手、小巧、易学
适合: 移动端项目，小巧
发展势头很猛，github上start数量超越别的框架

个人维护项目
不兼容低版本浏览器(ie8)

过滤器

因vue2 废除过滤器方法 所以需要自己定义

<div id="box">
    {{"mssSsg" | capitalize }}
</div>

filters: {
   capitalize: function(value) {
         if (!value) { // 判断是否有value
             return ""
         }
         value = value.toString() // 把value转换成字符串
         return value.charAt(0).toUpperCase() + value.slice(1) // 第一位大写 + 除第一位的字符串
     }
   }
