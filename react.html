react

react一个框架、mvc的框架（view层）

虚拟DOM 通过监听数据改变的状态来渲染页面

m –> 数据
v –> 视图
c –> 控制器

React的设计思想极其独特，属于革命性创新，性能出众。代码逻辑却非常简单。所以，越来越多忍开始关注和使用，认为它可能是将来web 开发的主流工具

组件化开发

可组合
可重用
可维护

用法

  1. 引入2个js库
    react , react-dom , browser

  2. 创建组件
    React.createElement(“标签”，{需要设置的属性},内容)

  3. 插入
    ReactDOM.render(组件，显示和i在哪个元素中，渲染完成的回掉函数)

jsx语法 -> js + xml

browser.js将jsx语法转化为javascript这样才会被浏览器正常解析

JSX是React编写组件的一种语法规范，可以看为是js的扩展，它支持将HTML和JS混写在一起，最后编译为常规的js，方便浏览器解析

创建组件开头 一定要大写

注意：如果有多个标签进行嵌套，那么语法只支持顶层有一个标签，如果要使用多个标签，那么应该将多个标签用一个顶层标签包裹。
