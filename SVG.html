位图和矢量图

位图： 是由一个个像素组成的

优点：
可以识别更多颜色 图片看起来更加细腻

位图图像与矢量图像相比更容易模仿照片似的真实效果搜索。位图图像的主要优点在于表现力强、细腻、层次多、细节多，可以十分容易的模拟出像照片一样的真实效果。由于是对图像中的像素进行编辑，所以在对图像进行拉伸、放大或缩小等到处理时，其清晰度和光滑度会受到影响。位图图像可以通过数字相机、扫描或PhotoCD获得，也可以通过其他设计软件生成 （大牛版）

缺点：

放大后 会失真

位图在放大到一定限度时会发现它是由一个个小方格组成的，这些小方格被称为像素点，一个像素是图像中最小的图像元素。在处理位图图像时，所编辑的是像素而不是对象或形状，它的大小和质量取决于图像中的像素点的多少，每平方英寸中所含像素越多，图像越清晰，颜色之间的混和也越平滑。计算机存储位图像实际上是存储图像的各个像素的位置和颜色数据等到信息，所以图像越清晰，像素越多，相应的存储容量也越大。（大牛版）

矢量图： 绘图图像，在数学上定义为一系列点与点之间的关系

优点：

矢量图可以任意放大或缩小而不会出现图像失真现象

缺点：

矢量图不容易制作色彩丰富的图像，绘制的图像不很真实，并且在不同的软件之间交换数据也不太方便

SVG的使用方法

<!--图片引入svg-->
<img src="1.svg" style="widht:300px; height:300px;">
<!--背景引入svg-->
<div style="width:300px; height:300px; background:url(1.svg) no-repeat"></div>
<!--框架引入svg-->
<iframe src="1.svg" width="300px" height="300px"></iframe>
<!--把svg添加到div中-->
<div id="div1">
    <svg xmlns="http://www.w3.org/2000/svg" width="100%" height="100%">
        <circle cx="200" cy="200" r="80" fill="skyblue"></circle>
    </svg>
</div>

<!--引用外部svg格式-->
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE svg PUBLIC "-//W3C//DTD SVG 1.1//EN"
"http://www.w3.org/Graphics/SVG/1.1/DTD/svg11.dtd">
<svg xmlns="http://www.w3.org/2000/svg">
<circle cx="200" cy="200" r="80" fill="skyblue"></circle>

SVG标签

circle 圆
cx x轴的坐标 中心点位置
cy y轴的坐标 中心点位置
r 半径
fill 设置背景颜色 transparent 透明
stroke 边框颜色
stroke-width 边框颜色宽度
stroke-opacity 边框颜色透明度

可以用style的格式去写

recl 矩形
width 宽
height 高
x 距离左边的位置
y 距离上边的位置
rx ry （可以写一个） 四个角的弧度

line 线
x1 y1 第一个点位置
x2 y2 第二个点位置

g 对g标签内元素进行操作

transform=”translate(0,0)” 位移 （貌似不能写百分比）

text 添加文字
x y 文字位置
font-size 文字大小
text-anchor middle 居中 start 左侧 end 右侧
上下居中的话 改y值
cursor: pointer 鼠标移入变为手型

image 添加图片
x y 图片的坐标
width 图片的宽度
height 图片的高度
xlink:href=”地址”
